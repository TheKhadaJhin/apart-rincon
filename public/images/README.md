# Imágenes

El logo ya está cargado como:

```txt
/images/logo.jpeg
```

Para cargar fotos reales de propiedades:
1. Copia las imágenes dentro de esta carpeta.
2. Usa nombres simples, por ejemplo:
   - depto-1-1.jpg
   - depto-1-2.jpg
   - depto-2-1.jpg
3. En el panel admin, en el campo "Imágenes por URL", escribe:
   - `/images/depto-1-1.jpg, /images/depto-1-2.jpg`
